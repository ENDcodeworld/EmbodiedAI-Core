## 版本说明

### 语义化版本

EmbodiedAI-Core 遵循语义化版本规范 (MAJOR.MINOR.PATCH):

- **MAJOR**: 不兼容的 API 更改
- **MINOR**: 向后兼容的功能新增
- **PATCH**: 向后兼容的问题修复

### 版本支持

- 最新版本获得积极支持
- 前一个版本获得关键安全修复
- 更旧的版本不再支持
